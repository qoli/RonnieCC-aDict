# HTML Templates

Reusable HTML/CSS templates for Xiaohongshu static images live here.

Template requirements:

- fixed 1080px width;
- default 1440px height unless the draft requires square output;
- no external network fonts or images;
- source screenshots must be local repo files;
- layout must survive long Chinese titles without text overflow;
- exported PNGs should be recorded in the matching draft or asset inventory.

Initial templates:

- [template-cover-v1.html](template-cover-v1.html): cover card with title, subtitle, proof points, and product screenshot.
- [template-workflow-v1.html](template-workflow-v1.html): step-by-step workflow card with one product screenshot and three steps.
