詞典查詢 · iOS · iPadOS · Apple platforms

一個安靜的閱讀詞典。

aDict 讓查詞保持貼近閱讀：快速入口、本地詞典，以及個人的 MDX / MDict 詞庫，不必因為每一個詞而被切換走。

App Store TestFlight beta 產品筆記

它為跨語言閱讀、收藏詞典、並希望查詞工具像基礎設施一樣穩定工作的人而設計。

功能

查詢應該貼近閱讀。

01

舒服的查詞

它從一個輕量 Mac 與 iOS 查詞工具開始，重視排版、深色模式，以及低摩擦的搜尋入口。

02

自己的詞典文件

使用 MDX / MDict 文件、MDD 資源與同名 CSS，不需要把長期收藏的詞庫轉成 App 專用格式。

03

閱讀資料

Words、查詞歷史、收藏條目、iCloud 文件與詞典文件都應被當作產品資料，而不是偶然的 UI 狀態。

04

增強層

新版包含可配置的 Intelligence Enrichment 欄位，能在查詞結果後追加聚焦的補充內容。

歷史

一個小工具正在變成可維護的系統。

2019

先讓查詞可讀

aDict 起初是一個簡單、順眼的查詞工具，服務編程、閱讀與日常用詞確認。

2021

使用者自己的詞庫

2.0 把產品推向自帶 MDX 文件、Words、iCloud 文件與 iPad 工作流。

2024

一次性買斷

2.4 取消訂閱，因為低頻但長期存在的詞典工具應該可靠地留在那裡。

2026

3.0 架構重寫

目前的重寫把查詞協議、Provider、WebView 渲染、本地文件與使用者資料拆開，並已準備公開 beta 分發。

系統

更清楚的本地詞典系統。

App 殼層

SwiftUI 重寫負責搜尋、詞典切換、收藏、設定、StoreKit 權益狀態，以及 Apple 平台體驗。

Providers

Youdao、MDict、StarDict 被拆成 provider packages，並輸出同一套 aDictProtocol response envelope。

詞典文件

本地或 iCloud 文件會被掃描成 MDict 與 StarDict 詞典，包含 MDX、MDD、同名 CSS 與 StarDict bundle files。

持久化

查詞歷史、收藏與 enrichment rules 使用 SQLite / GRDB 並通過 CloudKit 同步；生成結果 cache 則保留在本地。
