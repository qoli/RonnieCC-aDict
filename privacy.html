Privacy

Privacy policy.

Last updated: April 27, 2026.

Data collection

aDict is designed as a lookup and reading utility. The app does not require an account for normal dictionary lookup.

Dictionary data

Local dictionary files and lookup activity should remain part of the user's reading environment. This website does not receive or store those local dictionary files.

Website analytics

This landing page is a static GitHub Pages site. No custom analytics script is included in this initial version.

Contact

For privacy questions, use the public contact links on ronniewong.cc.
